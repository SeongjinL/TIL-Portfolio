# Typora(Markdown) Tutorial



## 설치 및 환경 설정 방법

- https://www.typora.io/ 설치 파일 다운로드. (현재는 유로 이므로 구버전 사용 가능)
  - https://www.typora.io/releases/all 구버전 다운로드가능한 사이트.
  - 구버젼의 설치 파일을 받고 설치.
  - 프로그램 실행 후 좌측 상단 `파일`에서  아래에 있는 `환경설정`클릭
  - **환경설정** 에서 **이미지** 클릭
    - 첫번째 `When insert` 메뉴 변경.
    - `./$(filename).assets 경로로 이미지 복사` 로 변경
    - 두번째로 `가능하다면 상대적 위치 사용` 를 클릭해서 적용 하였습니다.



### 제목을 만드는 방법!

#### 종류

- # 가장 큰 글씨

- ## 2번째로 큰 글씨

- ### 3번째로 큰 글씨

- #### 4번째로 큰 글씨

- 이런 식으로 `#` 을 이용하여 글씨를 강조 할 수 있습니다.

  - 이런 강조 글씨는 `ctr1`  + 1~6 으로도 설정이 가능
  
    

### 순서 없는 목록!

##### 순서가 없는 목록으로써 새로운 표식이 생겼습니다. 사용 방법은 `-` 로 표시 할 수 있습니다.

- 표식 테스트

- 표식 테스트

- 표식 테스트
  - 이런 식으로 표식 다음 `Enter` 를 누르고 `Tap`를 누른다면 작성 할 수 있습니다
  
    

### 텍스트 스타일링!

##### 여러 종류의 텍스트 스타일을 특수문자로 표현 가능 합니다.

- *italic*

- **bold**

- `inline code`
  - 이런 식으로 표현 할 수 있습니다.
  
    

### 텍스트 블록!

##### 텍스트블록을 사용해서 각 프로그래밍 언어를 표현 할 수 있습니다.

- 사용 방법은 *````사용할 언어`*를 작성 해주시면 됩니다.

  - 사용 예시

    - ```python
      def my_func():
          print('Hello')
      ```

      

### 표

##### 표를 사용하는 방법과 작성 내용.

- 표를 사용 하려면 `|` 이라는 특수문자를 이용해야 합니다 명칭은 **파이프** 라고 불립니다.

  - 표를 작성 할 때는 **|이름|나이|주소|전화번호| **이렇게 작성 합니다.

    - 사용 예시

    - |  이름  | 나이 |  주소  |  전화번호   |
      | :----: | :--: | :----: | :---------: |
      | 이성진 |  25  | 경기도 | 01000002222 |
